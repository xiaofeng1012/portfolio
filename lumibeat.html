技術特色與開發細節

原生 Python 開發

挑戰 Pygame 效能極限，不依賴 Unity 或 Unreal。從音符物理公式到異步資源加載，皆由 100% 原生 Python 邏輯封裝而成。

動態解析度自適應

自研 Layout Scaling 技術。透過動態比例因子計算，確保遊戲在 1366x768 至 4K 環境下，軌道與判定線始終保持黃金比例。

雙押提示線邏輯

強化高難度譜面讀譜體驗。系統自動偵測相同時間戳記的音符，並以動態中心座標繪製連接線，徹底解決視覺脫節感。

20 Combo 里程碑特效

兼顧效能與視覺的打擊反饋。每 20 連擊觸發一次雙側舞台粒子噴發，透過 Alpha 通道優化實現震撼的物理煙火特效。
