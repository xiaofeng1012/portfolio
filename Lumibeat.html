LumiBeat

基於 Python 數位訊號處理技術的自動化節奏遊戲

Python 3.x Pygame (引擎) Librosa (DSP) OpenCV (影像同步)

1. 核心技術：自動譜面生成 (Auto-chart Generation)

傳統遊戲需人工製譜，LumiBeat 利用 Librosa 進行音樂特徵提取，實現「丟入 MP3 即刻遊玩」。

  • Onset Detection： 偵測音訊中的起始能量爆發點作為打擊點。
  • Energy Envelope： 透過能量包絡分析副歌（能量高峰期），自動增加 Note 密度。
  • JSON Cache： 首次分析後產出快取檔，二次載入速度提升 90%。

2. 遊戲系統架構

多執行緒優化

使用 threading 將音頻分析與 UI 載入分離，解決了 Python 在處理大型音頻時產生的畫面凍結問題。

精確同步演算法

利用動態補償 delta_time 與音頻位置 get_pos() 進行校準，即使在低 FPS 狀態下也能精確判定打擊點。

3. 視覺特效與粒子系統

為了彌補 Python 在處理複雜 3D 的不足，我自行實作了基於 2D 物理的粒子系統：

  • StageLight System： 與 Combo 數連動，當 Combo 達標時觸發噴發效果。
  • Video Sync： 使用 OpenCV 逐影格讀取背景影片，並與 Pygame 進行混合同步。

4. 學習體悟

在這個專案中，我獨立解決了音訊同步、UI 適配、以及 Python 效能瓶頸等問題。這證明了我具備從零開發完整產品邏輯的能力，並能熟練運用資料結構（如 JSON 快取管理）與演算法來優化使用者體驗。
